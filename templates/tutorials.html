Promotions - How it works

Promotion is a feature by which we can send WhatsApp marketing messages to the targeted users.

Currently we have 4 Promotion trigger types, means for 4 use cases the promotions will be sent to the users. These are:

  • Yearly : Every year, at a certain time, this promotion will be sent to the targeted users.
    • Birthday
    • Anniversary
  • Menu Selected : This promotion will be sent to the users who have been ordered the selected menu(s) in their bookings.
  • Inactivity : You have to put how many days a user have to be inactive in order to get this promotion message.
  • Reservation Count : This promotion is specially for repeated & loyal customers who have made bookings multiple times in a restaurant.

In the Validity Period configuration, you have to set two dates, Valid from and Valid to, which represents from which date the promotion will start working, and until which date, the promotion will be halted.

To create a whatsapp chatbot, you will need these resources:

  • Twilio account
  • A WhatsApp sender in Twilio (Verified)
  • OpenAI API key

WhatsApp Chatbot - Update Sales level

You can update sales level very easily by going to the chatbot update section. Currently we are offering 5 sales level. These are:

  • Level 1 - Reservation only
  • Level 2 - Menu Rewards
  • Level 3 - Dish Prioritization
  • Level 4 - Personalized Recommendations
  • Level 5 - Promotions Module
